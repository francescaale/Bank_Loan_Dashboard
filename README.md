# Bank Loan Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze bank loan performance, monitor lending trends, and evaluate loan repayment behavior. The dashboard provides key insights into loan applications, funded amounts, repayments, customer demographics, and loan quality to support data-driven decision-making.

## Dashboard Features

- Total Loan Applications KPI
- Total Funded Amount
- Total Amount Received
- Good Loans vs Bad Loans Analysis
- Loan Status Distribution
- Monthly Loan Trends
- Regional Loan Performance
- Customer Demographics Analysis
- Purpose of Loan Analysis
- Interactive Filters and Slicers

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Dataset Used

Bank Analysis Dataset - https://drive.google.com/drive/folders/1wjjTBUg2SHXJQwVNjI5vHLk6DjI2W7y7

## Key Insights

- Analysed total loan applications, funded amounts, and repayments.
- Classified loans into Good Loans and Bad Loans based on loan status.
- Identified lending trends over time.
- Compared loan performance across regions and customer segments.
- Highlighted factors affecting loan approval and repayment performance.
